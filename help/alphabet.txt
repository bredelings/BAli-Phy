-A         [partitions:]alphabet            The alphabet.
--alphabet [partitions:]alphabet

The alphabet can be:
   'DNA', 'RNA', 'Amino-Acids', 'Numeric', 'Triplets', or 'Codons'

If no partitions are specified, then the alphabet is applied to
every partition.

You can specify the nucleotide alphabet for Triplets or Codons.
    'Triplets[DNA]'           'Codons[RNA]'

You can specify the genetic code Codons.
    'Codons[,standard]'       'Codons[RNA,mt-vert]'
The code can be 'standard' (the default), 'mt-vert', 'mt-invert',
 'mt-yeast', or 'mt-protozoan'

Examples:
    '--alphabet=DNA'
    '-A RNA'
    '--alphabet=1,3:DNA --alphabet=2:Codons'
