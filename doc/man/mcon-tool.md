# NAME

**mcon-tool** - Convert MCON files

# SYNOPSIS

**mcon-tool** _infile_ [OPTIONS]

# DESCRIPTION

Convert MCON files

# ALLOWED OPTIONS:
**-h**, **--help**
: produce help message

**--unnest**
: file with alignment to annotate

**--atomize**
: file with tree

**-O** _arg_ (=MCON), **--output** _arg_ (=MCON)
: output format (TSV or MCON)
